
# 기차 예매 앱

## 프로젝트 소개

이 프로젝트는 Flutter를 사용하여 개발된 간단한 기차 예매 앱입니다.

## 주요 기능

- 출발역과 도착역 선택
- 좌석 선택 및 시각화
- 예매 확인 프로세스

## 설치 방법

1. 이 저장소를 클론합니다:
   ```
   git clone https://github.com/enirobot/flutter_train_app
   ```
2. 프로젝트 디렉토리로 이동합니다:
   ```
   cd flutter_train_app
   ```
3. 필요한 패키지를 설치합니다:
   ```
   flutter pub get
   ```
4. 앱을 실행합니다:
   ```
   flutter run
   ```
